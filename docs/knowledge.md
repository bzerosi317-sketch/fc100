## 设计文档概览

### 核心功能模块

1. **数据看板** - 系统首页，展示设备、任务、告警的整体概览
2. **任务管理** - 任务全生命周期管理（创建、执行、监控、归档）
3. **航线管理** - 航线文件管理（导入、预览、编辑、归属组织）
4. **实时画面** - 多路视频监控、设备驾驶舱、地图位置展示
5. **媒体中心** - 任务产出照片的管理和查看
6. **杆塔台账** - 配网线路杆塔信息管理
7. **设备管理** - 无人机设备基本信息管理
8. **系统设置** - 平台配置、流媒体配置、人员管理、组织管理

### 接入设备类型

- **FC100 大载重无人机** - 物资投送、吊运任务
- **御3 行业版** - 巡检、勘察任务
- **M4TD 便携式机场** - 自动起降、航线执行

### 接入平台

- **大疆司运平台** - FC100 大载重无人机接入
- **上云API（司空2）** - 御3、机场设备接入
- **Skyway2 私有版** - 边缘节点桥接

## 术语表

| 术语 | 说明 |
|------|------|
| 勘察任务 | 基于预设航线的巡检、勘察任务 |
| 运载任务 | 大载重无人机执行的物资投送、吊运任务 |
| 驾驶舱 | 单台设备的集中监控和控制页面 |
| OSD | 屏幕显示叠加信息（高度、速度、电量等） |
| KMZ/KML | 航线文件格式，用于描述航点和飞行路径 |
